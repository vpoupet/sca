# SCA Syntax

VS Code syntax highlighting for `.sca` cellular automaton rule files.

This is a small local extension. To use it without publishing, symlink or copy
this directory into your VS Code extensions directory, then reload VS Code.

```sh
ln -s "$(pwd)/tools/vscode-sca-syntax" "$HOME/.vscode/extensions/local.sca-syntax"
```

Open any `.sca` file and VS Code should select the `SCA` language mode.
